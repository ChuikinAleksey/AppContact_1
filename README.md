Приложение для списка контактов на Android.
За основу брал: https://github.com/ishara-madu/ContactApp
Сделано:
- Загрузка контактов из телефонной книги устройства
- Сортировка контактов по алфавиту
- Звонок при нажатии на контакт
- Отображение списка
Из исходного проекта использована только идея структуры списка контактов. 
Стек: Kotlin, Android SDK, RecyclerView, CardView, ContentResolver, ContactsContract, Intent.ACTION_DIAL.
